# Testes

Este documento define as diretrizes gerais de teste.

## Testes por módulo

Cada módulo deve possuir:

- testes unitários de regras;
- testes de casos de uso;
- testes de integração com persistência;
- testes de contratos públicos;
- testes determinísticos para sistemas aleatórios.

## Testes do simulador

O simulador de partidas deve possuir também:

- testes de invariantes;
- testes estatísticos;
- testes de distribuição;
- testes de correlação;
- testes de calibração.
