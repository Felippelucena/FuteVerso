# ADRs

Decisões arquiteturais importantes devem ser registradas em ADRs.

## Estrutura sugerida

```text
docs/
├── DIRETRIZES_DESENVOLVIMENTO.md
└── adr/
    ├── 0001-monolito-modular.md
    ├── 0002-conteudo-e-campanha.md
    ├── 0003-comunicacao-por-eventos.md
    └── 0004-simulador-como-fronteira.md
```

## Cada ADR deve conter

- contexto;
- decisão;
- alternativas consideradas;
- consequências;
- status.
